

# Clinical Report Classification System

A BioBERT-powered NLP application that classifies clinical reports into 24 medical specialties.

## Features

- BioBERT Fine-Tuning
- Medical Specialty Classification
- Top-3 Predictions
- Confidence Scoring
- Readability Analysis
- PDF Upload Support
- FastAPI Backend
- Streamlit Dashboard

## Dataset

Medical Transcriptions Dataset
~4,900 Clinical Reports

## Model Performance

Accuracy: 83.6%

## Tech Stack

Python
PyTorch
BioBERT
FastAPI
Streamlit
HuggingFace Transformers
